# Contributing to Greek Data Science Roadmap

First off, thank you for considering contributing to this roadmap! The goal is to build the best, most up-to-date guide for aspiring Data Professionals in Greece.

## How Can I Contribute?

### 1. Add new Resources
If you know a great free/paid course, YouTube channel, or book that helped you, add it to the `resources.json` file.
- Fork the repo.
- Add a new JSON object to the array in `resources.json`.
- Create a Pull Request (PR) with a short description.

### 2. Update Salary & Market Data
The Greek job market is constantly evolving. If you have updated metrics or salaries, please submit a PR modifying the `README.md` file under the "Greek Job Market" section.

### 3. Fix Typos or Improve the UI
If you want to improve the interactive `roadmap.html` or fix typos in the README, please do so!

## Pull Request Process
1. Make your changes in a new branch.
2. Ensure your changes make sense and follow the existing structure.
3. Submit a Pull Request. I will review it as soon as possible.

Happy Coding! 🚀
